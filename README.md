# Nearby

# Sobre o projeto

**Nearby** é um aplicativo mobile de clube de benefícios com cupons para utilizar em estabelecimentos próximos a você.

Esse é o projeto do evento NLW Pocket Mobile, um dos conteúdos disponíveis para alunos da Rocketseat.

## Links dos outros repositórios

- Mobile https://github.com/EduardoAraD/nearby
- Backend https://github.com/EduardoAraD/backend-nearby

# Tecnologias utilizadas
## Mobile
- TypeScript
- React Native
- Expo
- Maps
- Qr Code

# Como executar o projeto
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/EduardoAraD/nearby.git

# instalar dependências
npm install

# executar o projeto
npx expo start
```
